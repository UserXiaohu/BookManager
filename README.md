# BookManager
【基于SSM的图书管理系统-有文档】

项目经过多人测试运行，可以确保100%成功运行。

### 项目简介
这是一个基于ssm的图书管理系统：
该系统具有二种权限：
管理员：1.图书管理
2.借书和还书操作
		3.书籍推荐
		4.读者增删改查
		5.借阅记录查询
		6.修改密码
读者：1.查询图书
	  2.个人借阅记录查询
	  3.个人信息修改
4.图书推荐
